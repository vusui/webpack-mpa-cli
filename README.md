# webpack-mpa-cli

### webpack构建多页应用前端项目模板(脚手架)

## 安装
```
$ npm install webpack-mpa-cli --save-dev
```

* 你可以在此项目中正常使用[vue](https://github.com/vuejs/vue)、[react](https://github.com/facebook/react)等前端框架

### 使用说明  

```bash
  #以下命令均可在package.json中进行配置

  #开发环境下运行
  npm run dev

  #使用Eslint进行代码检测
  npm run lint

  #打包构建
  npm run build

  #服务器环境下预览
  npm run serve
```
